## 从0到1的简历

### 简历模板
#### 联系方式
* 手机 
* E-mail 
* QQ/微信
#### 个人信息
* 姓名 
* 性别 
* 出生年份
* `Github`
* 技术博客
#### 教育经历
* 学院名称
* 绩点
* 相关课程
#### 求职意向 （非必须）
* 期待职位
* 期望城市
* 期待薪资
* 目前情况
#### 项目经历 （非必须）
#### 社团及组织经验 （非必须）
#### 致谢 （非必须）
感谢您花时间阅读我的简历，期待能有机会和您共事。
